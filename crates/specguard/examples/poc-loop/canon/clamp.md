# canon: clamp_score

スコア値を許容範囲に収める正典。実装は `src/clamp.py` の module-level 関数
`clamp_score(n)` として存在する。

規則 (いずれも反証可能):

- **R-LOW**: 入力 `n < 0` のとき、ちょうど `0` を返す。
- **R-HIGH**: 入力 `n > 100` のとき、ちょうど `100` を返す。
- **R-MID**: 入力 `0 <= n <= 100` のとき、`n` をそのまま返す。
- **R-INT**: 整数を受け取り整数を返す。float への暗黙変換はしない。

範囲は **0..=100 の閉区間**。境界値 `0` と `100` は範囲内 (clamp されない)。
